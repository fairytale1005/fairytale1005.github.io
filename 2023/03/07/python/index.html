2.7. range()

用于生成数字序列

  1. 语法range(num)
    1. 获取从0开始到num结束的序列(不包括num本身)
  2. 语法range(num1, num2)
    1. 获取从num1开始到num2结束的序列(不包括num2本身)
  3. 语法range(num1, num2, step)
    1. 获取从num1开始到num2结束的序列(不包括num2本身)
    2. 数字之间的步长，step默认为1

2.8. 函数

若无return函数默认返回None

2.8.1. 局部变量

函数内的变量声明均为局部变量，函数运行后就会被销毁，如果函数内部想修改全部变量，则需要关键字 global

2.8.2. 多返回值

  • 按照返回值的顺序，写对应顺序的多个变量接收即可
  • 变量之间用逗号隔开
  • 支持不同类型的数据return

2.8.3. 多种传参方式

  • 位置参数
    • 传递的参数和定义的参数的顺序及个数必须一致
  • 关键字参数
    • 函数调用时通过形参=值形式传递参数
    • 如果有位置参数时，位置参数必须在关键字参数的前面，但关键字参数之间不存在先后顺序
  • 缺省参数
    • 定义函数的时候可以定义形参=值的缺省参数
    • 所有位置参数必须出现在默认参数前，包括函数定义和调用
    • 函数调用时，如果为缺省参数传值则修改默认参数值, 否则使用这个默认值
  • 不定长参数
    • 位置传递
      • def pos(*args)
      • 传进的所有参数都会被args变量收集，它会根据传进参数的位置合并为一个元组(tuple)，args是元组类型
    • 关键字传递
      • def key(**kwargs)
      • 参数是“键=值”形式的形式的情况下, 所有的“键=值”都会被kwargs接受, 同时会根据“键=值”组成字典

2.8.4. 匿名函数

  • 函数作为参数传递
  • lambda匿名函数
    • lambda 传入参数:函数体
    • 函数体只能写一行

2.9. 序列切片

  • 序列：内容连续、有序、可使用下标索引的一类数据容器
  • 切片：从一个序列中取出一个子序列

列表、元组、字符串均可视作序列

表示从序列中，从指定位置开始，依次取出元素，到指定位置结束，得到一个新序列

切片不会影响序列本身，而是得到一个新的序列

2.9.1. 语法

序列[起始下标:结束下标:步长]

下标可以为正序或者反序，但要统一

3. 数据容器

为了批量存储或批量使用多份数据，根据是否支持重复元素、是否可以修改、是否有序分类

python五容器：list(列表)、tuple(元组)、str(字符串)、set(集合)、dict(字典)

  • 是否支持重复元素
    • 可重复：列表、元组、字符串
    • 不可重复：集合、字典
  • 是否可修改
    • 可修改：列表、集合、字典
    • 不可修改：元组、字符串
  • 是否有序（是否支持下标索引）
    • 有序：列表、元组、字符串
    • 无序：集合、字典

6. Python模块

6.1. 模块的导入

[from 模块名]import[模块|类|变量|函数|*][as 别名]

6.2. 自定义模块

每个Python文件都可以作为一个模块，模块的名字就是文件的名字. 即自定义模块名必须要符合标识符命名规则

6.2.1. _name_

如果在模块中有测试代码，无论是当前文件，还是其他已经导入了该模块的文件，在运行时都会自动执行测试代码。为了使测试代码只在当前文件中调用该函数，其它导入的文件中不执行，则在测试代码前添加if __name__ == '__main__':

当导入多个模块的时候，且模块内有同名功能. 当调用这个同名功能的时候，调用到的是后面导入的模块的功能

6.2.2. _all_

如果一个模块文件中有__all__变量，当使用from 模块名 import *导入时，只能导入这个列表中的元素

注意：__all__变量是一个列表

8.3. 类型注解

8.3.1. 功能

  • 帮助第三方IDE工具（如PyCharm）对代码进行类型推断，协助做代码提示
  • 帮助开发者自身对变量进行类型注释

8.3.2. 支持

  • 变量的类型注解
    • 方式1变量:类型
    • 元组 类型详细注解需要将每一个元素都标记
      • my_tuple : tuple[str, int, bool] = (“study”, 666, True)
    • 字典 类型详细注解需要两个类型key和value
      • my_dict : dict[str, int] = {“Math” : 99}
    • 方式2# type: 类型
      • var = random.randint(1, 10) # type: int
  • 函数（方法）形参列表和返回值的类型注解
    • def 函数方法名(形参:类型, ...) -> 返回值类型:
  • Union类型
    • Union[类型,..]

8.4. 面向对象三大特性

8.4.1. 封装

私有成员变量或者方法：名以__开头即可

只有类中其它成员可以访问私有成员

  • 为私有变量赋值：不报错但无效
  • 获取私有变量的值：报错
  • 使用私有方法：报错

8.4.2. 继承

8.4.2.1. 基本语法

多继承中，如果有同名成员，默认以继承顺序(从左到右)为优先级；先继承的被保留， 后继承的被覆盖

pass补全语法，确保一个类的完整，表示无内容

8.4.2.2. 复写和使用

复写：即在子类重新定义父类的成员即可，成员属性名或成员方法名不变

调用父类同名成员：父类名.xx 父类名.方法名(self)

​ 或者super().xx super().方法名()

8.4.3. 多态

同样的函数，传入不同对象，得到不同状态

抽象类：含有抽象方法的类(接口)
